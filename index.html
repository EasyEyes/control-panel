Threshold

Welcome to Threshold, a PsychoJS-based experiment generator designed to help you measure perceptual thresholds.

All you need to run your experiment is a few accounts (Pavlovia and maybe Prolific or Amazon Mechanical Turk) and a table that you create in any spreadsheet app, e.g. Google Sheets. The manual explains how to create your table and run your experiment. We offer two degrees of integration with Prolific. Full integration uses their API, which, because it's only in beta release, currently requires a token that you can only get by emailing Prolific to request one, https://prolificapi.docs.apiary.io/, which may take a few days. Alas. We can save your token for you in your Pavlovia account.

First, provide any needed consent form and font files that are not already in your Pavlovia account. Then provide your experiment table, which will be immediately checked for errors.
