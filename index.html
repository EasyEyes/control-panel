EasyEyes

Welcome to EasyEyes, a PsychoJS-based experiment compiler designed to help you measure perceptual thresholds online.

All you need to run your experiment is a few accounts (Pavlovia and maybe Prolific) and a table that you create in any spreadsheet app, e.g., Google Sheets. The Manual and Parameter Glossary explain how to create your experiment table, compile it into a participant web app, and run it. You can download our demo with all the files you need (table, fonts, forms, and a README of instructions) to run an experiment.

You can check your table without any account. Just submit it to the drop box. You can online test yourself, and people you personally recruit, with just a Pavlovia account. To use a recruitment service and test those participants online, you'll need both Pavlovia and Prolific accounts. We plan to offer MTurk support as well, as an alternative to Prolific. dgp
