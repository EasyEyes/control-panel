Threshold

Welcome to Threshold, a PsychoJS-based experiment generator designed to help you measure perceptual thresholds.

All you need to run your experiment is a few accounts (Pavlovia and maybe Prolific or Amazon Mechanical Turk) and a table that you create in any spreadsheet app, e.g. Google Sheets. The manual explains how to create your table and run your experiment.

  1. Pavlovia account
  2. Submit ↓ any missing fonts & consent/debrief forms.
    Drop files here, or click to browse for them.
  3. Submit ↑ your experiment table, to be checked now.
  2. In Pavlovia, hit the "RUNNING" button, then return here.
